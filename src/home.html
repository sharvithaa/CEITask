Lets see some Git commands and its uses

  1. git init - Initialize git repo
  2. git clone [repo url] - Create local copy of git repo
  3. git status - check status
  4. git add [filename] - add file to staging area
  5. git add -A - Add all files to staging area
  6. git commit -m "commit name" - Saving files to local repo
  7. git branch - list all branches
  8. git branch [branch name] - create a new branch
  9. git branch -d [branch name] - Delete the branch
  10. git cherry-pick - apply specific commit from one branch to another
